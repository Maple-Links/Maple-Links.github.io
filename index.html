want to learn more about maple?


Maple, (Mathematical. Analytical. Processing. Language.) is a lightweight, portable coding language made to replace harder languages such as c++ or c. Maple is also a free one-man-made coding language, the first version was made within the span of 2 months and based on the popular c++!

To write with MAPLE all you have to do is find the text editor of your choice, then save your code as a .txt file. Once your ready, upload a .txt or .maple file to the MAPLE Terminal and watch the magic happen.

The Maple 2nd edition package contains the Maple Terminal v1.1, and a guide book to Maple. To install the 2nd Edition Maple Package, go to the download page and install the package, then unzip the package, after that remove files from zip file and move it to a desired location.

To support MAPLE, create and share your own programs with the community. Maple is a 100% free non-profit coding language created 2023.
